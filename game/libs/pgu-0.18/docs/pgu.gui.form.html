class Form(pgu.gui.widget.Widget)
    A form that automatically will contain all named widgets.
 
After a form is created, all named widget that are subsequently created are 
added to that form.  You may use dict style access to access named widgets.
 
Example:
 
    f = gui.Form()
    
    w = gui.Input("Phil",name="firstname")
    w = gui.Input("Hassey",name="lastname")
    
    print(f.results())
    print('')
    print(f.items())
    print('')
    print(f['firstname'].value)
    print(f['lastname'].value)

Connect an event code to a callback function.
 
Note that there may be multiple callbacks per event code.
 
Arguments:

    fnc -- callback function
    *values -- values to pass to callback.  
 
Please note that callbacks may also have "magicaly" parameters.

    _event -- receive the event
